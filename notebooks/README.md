# CXRaide Notebooks

Notebook workspace for NIH and VinBig chest X-ray data.

- `DataPreparation/`: preprocessing, merging, and balancing notebooks.
- `ML/`: dataset export and model-development notebooks.
- `data/raw/`: trackable CSV metadata plus local-only raw image folders.
- `outputs/`: generated figures and local-only model artifacts.

Raw image folders are local-only and ignored by Git.
